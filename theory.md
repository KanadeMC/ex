# Ответы к экзамену ОП.04 Основы алгоритмизации и программирования

---

## Билет №1

**Вопрос 1:** Что такое алгоритм? Опишите три основные структуры алгоритмов (следование, ветвление, цикл).

**Ответ:** Алгоритм — это последовательность чётких инструкций для решения задачи. Три основные структуры:
- **Следование** — команды выполняются одна за другой.
- **Ветвление** — выбор одной из веток в зависимости от условия (if-else).
- **Цикл** — повторение блока кода, пока истинно условие (for, while).

**Вопрос 2:** Как в C# объявить метод? Что такое параметры и возвращаемое значение? Пример простого метода.

**Ответ:**
```csharp
тип_возврата ИмяМетода(параметры) { тело }
```
Параметры — входные данные, возвращаемое значение — результат работы. Пример:
```csharp
int Sum(int a, int b) { return a + b; }
```

## Билет №2

**Вопрос 1:** Объясните разницу между циклами for, while и foreach. Когда какой удобнее?

**Ответ:**
- **for** — когда известно количество итераций.
- **while** — когда нужно повторять, пока истинно условие (может ни разу не выполниться).
- **foreach** — для перебора коллекций/массивов без индекса.

**Вопрос 2:** Какие типы данных в C# вы знаете? Чем отличается int от float?

**Ответ:** int (целые), float/double (с плавающей точкой), char (символ), string (строка), bool (логический) и др.  
**int vs float:** int — целое число, float — дробное (с плавающей точкой, менее точный, чем double).

## Билет №3

**Вопрос 1:** Что такое условная конструкция if-else if-else? Пример проверки возраста (ребёнок, подросток, взрослый).

**Ответ:**
```csharp
if (age < 13) Console.WriteLine("Ребёнок");
else if (age < 18) Console.WriteLine("Подросток");
else Console.WriteLine("Взрослый");
```

**Вопрос 2:** Как организовать ввод числа с клавиатуры в C#? Как преобразовать строку в число (int.Parse или TryParse)?

**Ответ:** `int.Parse(Console.ReadLine())` — преобразует строку в число (выдаёт ошибку при некорректном вводе).  
**TryParse** — безопасный способ: `int.TryParse(Console.ReadLine(), out int x)` — возвращает true/false.

## Билет №4

**Вопрос 1:** Что такое список List\<T\> в C#? Как создать список, добавить элемент, получить длину?

**Ответ:** List\<T\> — динамический список.  
Создание: `List<int> list = new List<int>();`  
Добавление: `list.Add(10);`  
Длина: `list.Count`

**Вопрос 2:** Что такое переменная? Правила именования переменных в C# (camelCase).

**Ответ:** Переменная — именованная область памяти для хранения данных.

## Билет №5

**Вопрос 1:** Что такое операторы break и continue в циклах? Пример break.

**Ответ:** **break** — немедленно выходит из цикла. **continue** — пропускает текущую итерацию и переходит к следующей.  
Пример break:
```csharp
for (int i = 0; i < 10; i++) { if (i == 5) break; }
```

**Вопрос 2:** Как работает Console.WriteLine()? Как вывести несколько значений через запятую?

**Ответ:** Console.WriteLine() выводит строку в консоль и переводит на новую строку.  
Вывод нескольких значений: `Console.WriteLine($"{x}, {y}");` или `Console.WriteLine(x + ", " + y);`

## Билет №6

**Вопрос 1:** Что такое метод void? Чем отличается от метода с возвращаемым типом?

**Ответ:** void-метод не возвращает значение (просто выполняет действия).  
Метод с возвращаемым типом (int, string и т.д.) обязан вернуть значение через return.

**Вопрос 2:** Тернарный оператор ?: в C#. Пример.

**Ответ:** `условие ? значение_если_true : значение_если_false`  
Пример: `string result = (x > 0) ? "Положительное" : "Отрицательное";`

## Билет №7

**Вопрос 1:** Что такое отладка (debugging)? Способы отладки в Visual Studio / VS Code.

**Ответ:** Отладка — поиск и исправление ошибок в коде. Способы: точки останова (breakpoints), пошаговое выполнение (F10/F11), просмотр переменных (watch).

**Вопрос 2:** Что такое псевдокод? Зачем нужен перед написанием кода?

**Ответ:** Псевдокод — неформальное описание алгоритма на естественном языке без синтаксиса языка программирования. Нужен для проектирования логики до написания кода.

## Билет №8

**Вопрос 1:** Что такое тестирование? Чем отличается позитивный тест от негативного?

**Ответ:** Тестирование — проверка программы на корректность.  
**Позитивный тест** — проверка при корректных входных данных (должен работать).  
**Негативный тест** — проверка при некорректных данных (должен обработать ошибку).

**Вопрос 2:** Основные признаки хорошего алгоритма.

**Ответ:** однозначность, результативность, конечность, массовость, дискретность.

## Билет №10

**Вопрос 1:** Что такое рефакторинг? Приведите простой пример улучшения кода.

**Ответ:** Рефакторинг — изменение кода без изменения его поведения, чтобы улучшить читаемость/структуру.  
Пример: заменить повторяющийся код методом, переименовать переменную в понятное имя.

**Вопрос 2:** Комментарии в C#: //, ///, /* */. Зачем нужны?

**Ответ:**
- `//` — однострочный
- `///` — XML-комментарий для документирования
- `/* */` — многострочный
Нужны для пояснения кода, но хороший код должен быть понятен без комментариев.

```csharp

```

## Билет №17

**Вопрос 1:** Локальные и глобальные (статические) переменные в C#. Пример.

**Ответ:** Локальные — объявлены внутри метода, видны только в нём.  
Глобальные (статические) — объявлены в классе со static, доступны из любого метода.  
Пример: `static int count = 0;` — глобальная, `int x = 5;` внутри метода — локальная.

**Вопрос 2:** Оператор Contains для списка или строки.

**Ответ:** Contains проверяет, содержится ли элемент в списке или подстрока в строке.  
Пример: `list.Contains(5)` / `str.Contains("abc")`.

## Билет №21

**Вопрос 1:** Что такое флаг (переменная-индикатор)? Пример использования в цикле.

**Ответ:** Флаг — bool-переменная, отмечающая выполнение условия. Пример:
```csharp
bool found = false;
for (int i = 0; i < arr.Length; i++) {
    if (arr[i] == target) { found = true; break; }
}
```

**Вопрос 2:** Оператор return в цикле. Пример досрочного выхода.

**Ответ:** return в цикле завершает и метод, и цикл досрочно:
```csharp
int FindIndex(int[] arr, int val) {
    for (int i = 0; i < arr.Length; i++)
        if (arr[i] == val) return i;
    return -1;
}
```

## Билет №22

**Вопрос 1:** Сортировка пузырьком. Принцип работы.

**Ответ:** Проход по массиву, сравнение соседних элементов и их обмен, если они в неправильном порядке. Крупные элементы "всплывают" в конец. Повторяется, пока массив не отсортирован.

**Вопрос 2:** Как скопировать массив? Чем отличается b = a от b = (int[])a.Clone()?

**Ответ:**  
`b = a` — копирует ссылку (оба указывают на один массив).  
`b = (int[])a.Clone()` — создаёт новый массив-копию (независимый).

## Билет №23

**Вопрос 1:** Линейный поиск в массиве. Алгоритм.

**Ответ:** Проходим по массиву от начала до конца, сравнивая каждый элемент с искомым. Если нашли — возвращаем индекс, иначе -1.

**Вопрос 2:** Логические операторы &&, ||, !. Примеры.

**Ответ:**
- `&&` — И (оба true)
- `||` — ИЛИ (хотя бы один true)
- `!` — НЕ (отрицание)
Пример: `if (x > 0 && x < 100)`

## Билет №24

**Вопрос 1:** Что такое двумерный массив (int[,]) в C#? Как обратиться к элементу a[i,j]?

```csharp

        Console.Write(matrix[i, j] + " ");
```

## Билет №26

**Вопрос 1:** Оператор % (остаток от деления). Примеры применения.

**Ответ:** Возвращает остаток от деления.  
Применение: проверка чётности (`x % 2 == 0`), выделение цифр числа.
