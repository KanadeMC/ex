# Решения для заданий из практической части. Экзамен по 
## Как пользоваться
Все билеты написаны в Z{Номер задания}.P{Часть практики}. 
> **Пример: Билет 1 задание 3** находятся в **Z1.P1**
## Проверка заданий
Чтобы проверить задание достаточно поменять 2 числа в Program.cs<br>
> **Пример: если хотим проверить 5 билет задание 4** то напишем:
> ```csharp
> Z5.P2 part = new();
> part.Run();
> ```
